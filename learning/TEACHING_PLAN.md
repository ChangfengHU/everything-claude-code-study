## 🎯 演示项目准备

为了教学，我们将创建一个 `practice` 项目，包含：

```
everything-claude-code-study/
└── practice/
    ├── calculator/          # 简易计算器 (Go)
    ├── todo-api/           # 待办事项 API (Node.js)
    └── web-scraper/        # 网页爬虫 (Python)
```

每个项目都有不同用途：
- **calculator**: 演示 TDD、Go patterns
- **todo-api**: 演示 API 设计、安全审查
- **web-scraper**: 演示代码重构、清理
