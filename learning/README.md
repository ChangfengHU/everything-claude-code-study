# Everything Claude Code 学习项目

## 项目简介

这是一个完整的学习项目，用于演示和记录 `everything-claude-code` 插件的所有功能。

> **Everything Claude Code** 是来自 Anthropic 黑客松获胜者的完整 Claude Code 配置集合，包含经过实战验证的 agents、skills、hooks、commands、rules 和 MCP 配置。

---

## 学习模块

| 模块 | 目录 | 内容 |
|------|------|------|
| **01. Commands** | `./01-commands/` | 斜杠命令 - 快速执行常用工作流 |
| **02. Agents** | `./02-agents/` | 专门子代理 - 委托特定任务给专业 AI 助手 |
| **03. Skills** | `./03-skills/` | 工作流技能 - 定义编码模式和最佳实践 |
| **04. Hooks** | `./04-hooks/` | 自动化触发器 - 基于事件的自动化操作 |
| **05. Rules** | `./05-rules/` | 编码规范 - 始终遵循的开发准则 |

---

## 学习路径

建议按顺序学习每个模块，从最常用的 Commands 开始：

```
01-commands → 02-agents → 03-skills → 04-hooks → 05-rules
```

每个模块目录下都有详细的说明文档和演示示例。
